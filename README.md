# Cockpit

High-tech Tony Stark-style macOS control plane and information display layer.

## Goals
- Advanced 3D / holographic UI
- Always-on ambient awareness (camera + mic)
- Local inference integration (MLX + Ollama)
- Cross-platform (Mac native + web)

## Current Status
- Project structure initialized
- Work in progress on core architecture

## Folder Structure
- `Sources/Cockpit/App/` — App entry point and lifecycle
- `Sources/Cockpit/Views/` — SwiftUI views
- `Sources/Cockpit/Models/` — Data models
- `Sources/Cockpit/Services/` — Business logic and services
- `Sources/Cockpit/3D/` — 3D / holographic components
- `Sources/Cockpit/Resources/` — Assets and resources
